About EyePredict

EyePredict leverages deep learning to assist ophthalmologists by analyzing retinal fundus images and identifying diseases such as glaucoma, diabetic retinopathy, and macular degeneration. The integrated Grad-CAM visualization highlights the critical regions used by the AI model for interpretability and trust.

Built using FastAPI and TensorFlow, this project merges state-of-the-art deep learning with modern web technologies for accessible AI diagnostics.
